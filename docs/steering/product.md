# プロダクト概要

* 製品名: skillmap
* ミッション / 目的: Skyrim風スキル星座をWebで操作できるプロトタイプとして技能管理体験を提供
* ターゲットユーザー:
  * Skyrim風のスキルUIを試したいゲーマー・ファン
  * ブラウザ完結のスキル管理UIを検証したいフロントエンド開発者
* 主要機能（簡潔に箇条）:
  * スキル星座の可視化・操作: Vue 3コンポーネントでスキルの位置や接続を表示
  * スキルポイント管理: 残ポイントと解放済みスキルIDをブラウザ内に保存
  * データ保存: Service Worker + OPFS でのローカル永続化（外部APIなし）
  * 共有・出力: QRコード生成依存を利用した共有想定
* ビジネス目標 / 成功指標:
  * KPI例: 主要スキル解放シナリオの完了率やローカル保存の成功率
* 実装状況サマリ:
  * 現在の完成度: フロントエンドはVue 3 + Viteで実装済み、バックエンド連携は廃止しブラウザ内完結に移行
